カフェ 雫-coffee drip-様

カフェ 雫-coffee drip- 公式サイト

ハンドドリップやサステナブルな取り組みにこだわるカフェ「雫 -shizuku coffee drip-」様の、ブランドの世界観を丁寧に伝えるWebサイトを制作しました。
来店促進やSNSフォロー獲得を目的に、構成・導線設計からデザイン・実装まで一貫して担当しています。

サイト種別 コーポレートサイト
課題 店舗のこだわりや想いを伝える手段がなく、新規のお客様に魅力が届きづらい。
目的 コーヒー豆や店舗空間に込めた世界観・価値観を正しく伝え、ブランドの理解・共感・認知を広げる。 Webを通じて来店やSNSフォローへとつなげ、 今後の展開に向けた土台を整備する。
ターゲット カフェ文化やライフスタイルに関心のある30〜40代女性を中心に、世界観や理念に共感する感度の高い層。
コンセプト 「丁寧に淹れた一杯と静寂の空間で、日常をリセットする上質なカフェ体験。」
ただの情報提供ではなく“体験”を届けるサイトに。
ポイント トップで「しずく」や「波紋」をモチーフにしたアニメーションを実装し、体験の入口としての印象を強化 色彩や余白、動きに“静けさ”と“自然なゆらぎ”を取り入れ、ブランドの空気感を視覚化 構成・UI設計は初見ユーザーでも流れが掴みやすいよう配慮し、回遊性と理解促進を両立
制作期間 構成設計：約1週間
ワイヤー作成・デザイン：約1週間
コーディング：約2週間
使用ツール Figma / VSCode HTML / CSS / JavaScript / WordPress
